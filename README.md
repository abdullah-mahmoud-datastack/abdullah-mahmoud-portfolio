## ✨ Key Features

This portfolio is custom-built from scratch to reflect my engineering approach—clean, modular, and performance-conscious.

### 🎨 Design & UX

- **Fully Responsive**: Optimized for Mobile, Tablet, and Desktop screens.
- **Dark/Light Mode**: User-toggled theme preference with smooth transitions.
- **Interactive UI**: Smooth scroll animations, fade-in effects, and interactive cards.
- **Performance-conscious**: Semantic HTML5, maintainable CSS variables, and zero heavy frameworks.

### 📂 Dedicated Sections

- **Projects (`/templates/projects.html`)**: Detailed breakdowns of my data engineering pipelines and cloud architectures.
- **Certificates (`/templates/certificates.html`)**: Categorized credentials (Data Engineering, AI & ML, Data Visualization) with verification links.
- **Courses (`/templates/courses.html`)**: A log of ongoing and completed technical courses.
- **Books (`/templates/books.html`)**: A curated reading list covering data engineering, architecture, and personal development.
- **Netlify Contact Form**: Fully functional serverless contact form embedded on the home page.

## 🚀 Local Development

This website is a static site and requires no complex build tools to run locally.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/abdullah-mahmoud-de/abdullah-mahmoud-portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd abdullah-mahmoud-portfolio
   ```
3. **Run locally:**
   - Open `index.html` directly in any modern web browser.
   - Alternatively, use a live server (e.g., VS Code Live Server extension) for hot-reloading:
     ```bash
     # If you have Python installed:
     python -m http.server 8000
     ```
